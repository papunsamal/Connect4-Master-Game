A modern and interactive Connect 4 game built using HTML, CSS, and JavaScript. The game features a responsive design, smooth animations, winner detection, draw detection, confetti celebration effects, and a modern UI.

## ✨ Features

* 🎯 Connect 4 game logic
* 🔴 Red vs 🟡 Yellow player turns
* 🏆 Winner detection
* 🤝 Draw detection
* 🎉 Confetti celebration effect
* 🔄 Restart game functionality
* 📱 Fully responsive design
* 🌈 Animated gradient background
* 💫 Glowing animated side orbs
* 🎨 Modern glassmorphism UI

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Connect4-Master-Game.git
```

2. Open the project folder

```bash
cd Connect4-Master-Game
```

3. Run the project

Open `index.html` in your browser.
